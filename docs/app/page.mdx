## Quick Example

```bash
curl -H "X-Api-Key: YOUR_API_KEY" \
  "http://localhost:8080/v1/tides?station_id=8454000&start=2024-01-01T00:00:00Z&end=2024-01-02T00:00:00Z"
```

Response:

```json
{
  "data": [
    {
      "station_id": "8454000",
      "time": "2024-01-01T00:00:00Z",
      "water_level_m": 1.234
    }
  ],
  "meta": {
    "count": 1,
    "source": "NOAA CO-OPS",
    "credits": "Data provided by NOAA"
  }
}
```

## Available Endpoints

- **Tides** - Water level measurements from tide stations
- **SST** - Sea surface temperature data
- **Currents** - Ocean current velocity vectors
- **Turbidity** - Water clarity measurements
- **Bathymetry** - Depth tiles for mapping
