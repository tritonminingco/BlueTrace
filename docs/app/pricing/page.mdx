## Metered Billing

Pro and Enterprise plans use metered billing through Stripe. You're charged based on actual API usage:

- **Pro**: $0.001 per API call beyond base plan
- **Enterprise**: Custom pricing based on volume

## Fair Use Policy

The Free plan is subject to fair use policies:

- Maximum 43,200 requests per day
- No commercial usage
- Best effort availability

## Questions?

- All plans include access to all datasets (tides, SST, currents, turbidity, bathymetry)
- Upgrade or downgrade anytime
- Cancel anytime, no lock-in
- Unused credits don't roll over

Contact us at sales@bluetrace.dev for custom plans or questions.
